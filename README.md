# Flask-Market


Simple Online market application.

Language: Python, SQL

Framework: Flask

Database: Mysql

ORM: Flask-SQLAlchemy

Front-end: Bootstrap

API urls:

api/ver0.1/items (GET)

api/ver0.1/items/ (GET, POST, PUT, DELETE)

Note: API works directly with MySQL.

Note: Ti use API, first login to market.
